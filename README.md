# Elm Develop Environment

[![CircleCI](https://circleci.com/gh/ababup1192/elm-dev-env.svg?style=svg)](https://circleci.com/gh/ababup1192/elm-dev-env)
## Install

```shell
$ yarn
```

## Launch develop server

```shell
$ yarn start
```

## Test

```shell
$ yarn test
```

## Build

```shell
$ yarn build
$ open dist/index.html
```
